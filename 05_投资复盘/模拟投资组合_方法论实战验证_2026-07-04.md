## 二、资产配置规则

### 基础规则

#### 规则1：基于方法论的决策矩阵

| 注水概率 | 真实就业强 | 真实就业弱 | 真实就业持平 |
|---------|----------|----------|------------|
| **< 30%** | ✅ 股票+10% | ⚠️ 股票-10% | ⚠️ 中性 |
| **30-60%** | ⚠️ 股票+5% | ❌ 股票-15% | ⚠️ 现金+10% |
| **> 60%** | ⚠️ 股票-5% | 🚨 股票-25% | ❌ 股票-10% |

#### 规则2：跨资产再平衡

```
情景：通胀高 + 就业弱（滞胀）
操作：
  - 股票 -10%
  - 现金 +10%
  - 黄金 +5%
  - 长期国债 -5%

情景：经济强 + 通胀低（金发女孩）
操作：
  - 股票 +10%
  - 长期国债 -10%
  - 现金 -10%
  - 黄金 -5%

情景：衰退担忧
操作：
  - 股票 -15%
  - 长期国债 +15%
  - 现金 +10%
  - 黄金 +5%
```

#### 规则3：套利窗口操作

数据公布后 T+1~5 套利窗口：

| 操作类型 | 标的 | 时点 |
|---------|------|------|
| 数据公布后做多 | 风险资产 | T+1 |
| T+5 减仓 | 风险资产 | T+5 |
| T+30 验证 | 全组合 | T+30 |

#### 规则4：风险控制

| 规则 | 限制 |
|------|------|
| **单资产最大占比** | 30% |
| **单日最大变动** | ±5% |
| **月最大变动** | ±15% |
| **止损线** | -10% 单笔 |
| **最大回撤** | -20% 整体 |

## 七、方法论验证

### 验证框架

通过模拟投资组合，验证方法论的**实战有效性**：

| 验证维度 | 方法 | 目标 |
|---------|------|------|
| **预测准确度** | 注水量预测 vs 实际下修 | >70% |
| **方向判断** | 市场反应 vs 实际走势 | >70% |
| **时机把握** | 套利窗口操作 vs 实际收益 | >0% |
| **风险控制** | 最大回撤 vs 预期 | <20% |
| **收益表现** | 总收益 vs 基准 | >基准 |

### 验证周期

| 周期 | 时间 | 验证内容 |
|------|------|---------|
| **周度** | T+7 | 短期操作验证 |
| **月度** | 每月末 | 月度策略验证 |
| **季度** | 季末 | 方法论整体验证 |
| **年度** | 12/31 | 全年综合验证 |

### 验证指标

| 指标 | 计算 | 目标 |
|------|------|------|
| **胜率** | 盈利交易 / 总交易 | >60% |
| **盈亏比** | 平均盈利 / 平均亏损 | >1.5 |
| **夏普比率** | (收益-无风险)/标准差 | >1.0 |
| **最大回撤** | 历史最大跌幅 | <20% |
| **基准超额** | 总收益 - 基准收益 | >0 |
