# 状态色标说明（文件夹之上）

> **时时变源：** [[90_系统配置/folder_status.yml|folder_status.yml]]（经 [[90_系统配置/文件夹状态]] 入口）  
> 改 `status` 后必跑：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File "D:\AI  金融知识点\90_系统配置\sync_folder_status_colors.ps1"
```

左侧文件夹会换成**柔和左边线**（进行中 = 蓝，不是刺眼黄）。CSS 由脚本生成，勿手改。

---

## 一、五色

| 色标 | 含义 | 文件夹视觉 |
|------|------|------------|
| 🟢 | 稳态 | 柔和绿左边线 |
| 🔵 | **进行中** | **柔和蓝**左边线 |
| 🟠 | 日程待办 | 柔和琥珀 |
| 🔴 | 阻塞 / 逾期 | 柔和红 |
| ⚪ | 暂停 | 灰 |

文档内进行中用 `[!info]` + 🔵，**不要**再用 `[!warning]` 金黄表示进行中。

---

## 二、时时变化流程

```text
编辑 folder_status.yml 的 status 字段
        ↓
运行 sync_folder_status_colors.ps1
        ↓
CSS + 下方看板自动更新
        ↓
必要时重载 Obsidian / 开关一次 CSS 片段
```
