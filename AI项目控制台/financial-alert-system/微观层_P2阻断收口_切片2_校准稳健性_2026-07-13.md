## 交付

| 项 | 说明 |
|---|---|
| `MicroCalibrator.rollingOutOfSample` | 走步样本外折；一致性率与符号翻转计数 |
| `MicroCalibrator.nonOverlappingOutcomes` | 按持有期时间净化 |
| `MicroCalibrator.costPressure` | 默认 10bps 成本发网后期望 |
| `buildRecord(...).robustness` | 每条记录附带三块证据；`requireRollingOos` / `requireCostStress` 可选硬门禁（默认关闭，不静默降级已有 validated） |
| `npm run inspect:micro-cal` | 输出摘要 + `artifacts/micro_calibration_robustness.json` |
| `npm run inspect:micro-cal:enrich` | 为既有 book 挂载 robustness（不升级 validated） |
| `smoke:micro-cal` | **11/11** |

## 诚实结论（证据，非放行）

对当前 6 条 `validated` 记录（summary scaffold 富集后）：

| 键 | rolling_oos | non_overlap kept | cost@10bps |
|---|---|---:|---|
| SOL any 48h/72h | fail (~0.33) | 有净化 | **fail**（净期望转负） |
| MULTI * 48h/72h | fail (~0.33–0.67) | 有净化 | **pass** |

因此：校准自动加载与证据管线已具备；**科学性强化未宣称通过**。研究级 RC 仍须在复验中诚实标注该缺口；不得因挂上 robustness 字段就升级执行权限。
