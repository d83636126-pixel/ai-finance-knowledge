# 微观层独立目录迁移（2026-07-13）

## 结论

微观层已从首页及 `static/`、`scripts/` 的散落文件，提升为独立功能目录：

```text
F:\financial-alert-system\micro-layer\
├─ index.html
├─ micro_layer.js
├─ micro_dashboard.js
├─ README.md
└─ tests\
   ├─ smoke_micro_layer.js
   └─ smoke_micro_page.js
```

## 职责

- `index.html`：微观层独立页面。
- `micro_layer.js`：归一化、数据质量、状态节点、因子聚合、市场状态、历史校准门禁、组合约束和弃权逻辑。
- `micro_dashboard.js`：页面渲染、缓存、实时刷新、文件模式边界。
- `README.md`：运行方式、目录职责和证据边界。
- `tests/`：计算边界与页面结构测试。

## 兼容处理

- 首页脚本改为加载 `micro-layer/micro_layer.js`。
- 首页“微观层”入口改为 `micro-layer/index.html`。
- 原 `micro.html` 保留为兼容跳转页，旧书签不会失效。
- 原 `static/micro_layer.js`、`static/micro_dashboard.js`、`scripts/smoke_micro_layer.js`、`scripts/smoke_micro_page.js` 已删除，避免双份代码漂移。
- `package.json` 的专项测试命令已切换到新目录。
