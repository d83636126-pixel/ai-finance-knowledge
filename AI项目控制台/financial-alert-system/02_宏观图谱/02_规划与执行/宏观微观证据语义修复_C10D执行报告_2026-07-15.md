## 模式契约

| mode | 含义 |
|---|---|
| `live_refresh_ok` | 有密钥且观测新鲜，可作 live 确认候选 |
| `last_good_snapshot` | 刷新失败后保留最近已知值；可写报告，不可 live 确认 |
| `snapshot_stale` | 有观测但超龄；降级 |
| `refresh_failed` | 刷新失败且无可恢复观测 |
| `no_key` | 当前 env/.env 无可用密钥 |
| `no_data` | 有密钥但无槽位数据 |

硬规则：

```text
历史 market_snapshot.has_fred_api_key 不参与 availability
mode != live_refresh_ok → cannot_treat_as_live_confirm = true
```
