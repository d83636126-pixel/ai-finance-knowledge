# 第三层验证指标

## 背景

上一轮已经补完美元、美债、实际利率、黄金、BTC 的二级传导：

```text
财政供给 -> 期限溢价 -> 美债收益率 / 美债回报 / Nasdaq
政策预期 -> 实际利率 / 美元
实际利率冲击 -> Nasdaq / Gold / BTC
美元压力 -> 风险资产 / 美债回报
```

本轮目标是给这些二级机制接入可观察的验证指标，让图谱不只会推演，还能回答：

```text
这条链现在有没有被市场数据确认？
```

## 设计说明

这些节点属于 verifier，不是新的中心目标。

它们的作用是给路径报告提供确认点：

- 美债链看：期限溢价、拍卖需求、MOVE。
- 实际利率链看：10Y TIPS。
- 美元链看：DXY 动量。
- 黄金链看：黄金 ETF 资金流。
- BTC 链看：BTC ETF 资金流。
- Nasdaq 链看：市场宽度。
