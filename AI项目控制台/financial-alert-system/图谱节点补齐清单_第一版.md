## 中心目标

已确认：

```text
中心目标 = 全球风险状态
输出目标 = 资产篮子

```

## 补齐原则

第一版不追求节点越多越好，而是优先补能解释全球风险状态和核心资产篮子的节点。

节点必须满足至少一条：

- 能影响全球流动性。
- 能影响美元指数。
- 能影响美债收益率或实际利率。
- 能影响风险偏好。
- 能影响 BTC / Nasdaq / Gold / USD / Treasury。
- 有可观察数据或事件来源。
