# ADR：暂停补样；冻结 Model v2 失败基准；仅并行 AR-S0；预注册 Model/Gate v3

## 决策

1. **暂停**向 Gate v2 机械补样。  
2. 将现有 `research_counted=20` 永久冻结为 **Model v2 真实失败基准**（禁止改卡刷分、禁止降门槛过关）。  
3. 立即完成 Gate v2 缺口诊断，并预注册 **Model/Gate v3**（两阶段 tradability→direction）。  
4. 架构并行范围 **仅 AR-S0**；AR-S1/S2/S3 等 v3 输入输出与评分协议冻结后再启动。  
5. v3 的 `RESEARCH_PASS` **不得**在同一批 20 样本上调参后宣称通过；须留出历史或真实前瞻验收。

## 理由

Gate v2 在 n≥20 后仍失败：方向 32%、Brier 0.742、Δ基线 −0.047；且模型与 `surprise_sign_only` / `no_propagation_graph` Brier 完全相同 → 问题在规则与校准，不是样本数。降低门槛会破坏门禁可信度。全面抽离 Graph/Scenario 会固化未证明的研究逻辑。

## 后果

- 开发集：冻结 20 场（诊断/对照）  
- 验收集：新历史留出或 live forward  
- 工程：70% v3 设计 / 20% AR-S0 / 10% 前瞻观察
