## 2. 首屏五问映射

1. 现在发生了什么 ← 正式流标题摘要  
2. 重大新闻及来源 ← 可点选正式条目（source / published_at / freshness）  
3. 市场/微观反应 ← 验证器条（live/restored/unavailable）+ 微观 iframe  
4. 宏观与传导 ← 同 `event_id` 跳转 macro / transmission / evidence  
5. 可信度与下一步 ← 正式条数 + 源降级计数；不足则明示不输出确定判断  

正式事实 vs 研究发现分栏；fixture/缺字段进非正式列。

## 3. 新闻流

- `loadNewsBundle`：Inbox → RealtimeTruth 评估 → 聚合正式流  
- 缺 `published_at` / demo / fixture → 非正式，展示拒绝原因  
- 选中写入 URL `event_id`/`as_of`，证据/宏观/微观/传导保持上下文
