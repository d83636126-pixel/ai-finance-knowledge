# UI Target 分层展示 2026-07-05

## 背景

图谱数据已写入：

```text
target_tier
target_priority
```

需要让前端 UI 能利用这些字段，优先展示核心 target，并区分扩展 target。

## 本次实现

已更新：

```text
F:\financial-alert-system\propagation.html
```

实现内容：

- 新增 `targetTierLabel()`。
- 新增 `targetSort()`。
- 图谱节点数据携带 `target_tier` 和 `target_priority`。
- 核心 target 节点在图上更大、白色边框、更亮。
- 扩展 target 节点使用绿色边框。
- 节点 tooltip 显示“核心 target / 扩展 target”和优先级。
- 逆向归因目标下拉框按 optgroup 分为：
  - 核心 target
  - 扩展 target
  - 其他 target
- 下拉框按 `target_priority` 排序。
