# 图谱节点扩展：全球非美国节点

> [!info] 历史别名
> 本文件与 [[AI项目控制台/financial-alert-system/图谱节点扩展_全球非美国节点]] 内容重复。主文件以后者为准；本文件仅保留旧链接兼容。

## 背景

当前 `financial-alert-system` 的图谱节点大部分围绕美国宏观、货币政策、美元流动性和风险资产展开。

用户提出：对世界经济影响很深的不只有美国，还应考虑其他关键经济体和全球系统性变量。

## 需求变化

传播图需要从“美国中心图谱”扩展为“美国主轴 + 全球关键外部冲击节点”。

## 推荐新增节点组

### 1. 中国节点

- 中国 PMI
- 中国 CPI / PPI
- 中国社融 / 信贷
- 中国房地产压力
- 人民币汇率
- 中国出口数据
- 中国政策刺激
- 中国股市风险偏好

### 2. 欧元区节点

- ECB 利率决议
- 欧元区 CPI
- 欧元区 PMI
- 德国工业数据
- 欧元汇率
- 欧洲能源压力
- 欧债利差

### 3. 日本节点

- BOJ 利率政策
- 日本国债收益率
- 日元汇率
- 套利交易 unwind
- 日本通胀

### 4. 全球美元融资节点

- DXY 美元指数
- 全球美元流动性
- 离岸美元融资压力
- 美元互换基差
- 跨境资本流动

### 5. 能源与商品节点

- 原油价格
- 天然气价格
- 铜价
- 黄金
- 农产品价格
- OPEC 供给冲击

### 6. 地缘与贸易节点

- 中美贸易冲突
- 俄乌冲突
- 中东冲突
- 红海 / 苏伊士航运扰动
- 关税政策
- 制裁政策

### 7. 全球风险资产节点

- MSCI 全球股指
- 新兴市场股指
- 全球信用利差
- VIX
- BTC
- 黄金

## 设计原则

非美国节点不应简单堆数量，而应符合以下标准：

- 是否能影响全球流动性。
- 是否能影响美元、利率、能源、商品或风险偏好。
- 是否能传导到 Nasdaq / BTC / 美债 / 黄金等目标资产。
- 是否有可观测数据。
- 是否能被验证或反驳。

## 下一步

- 按区域建立节点组：美国、中国、欧元区、日本、全球商品、地缘风险。
- 为每个节点标记 `line` 或 `lines`。
- 区分事实节点、解释节点、验证节点、目标节点。
- 优先补充能影响 BTC / Nasdaq / USD / Treasury / Gold 的节点。
