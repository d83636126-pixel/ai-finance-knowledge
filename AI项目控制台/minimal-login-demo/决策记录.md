# minimal-login-demo：决策记录

### 2026-07-05 保护 Lovable 项目历史

- 问题：项目连接 Lovable，Git 操作是否受限
- 选项：A) 正常使用 Git / B) 限制 force push / rebase / amend / squash
- 选定：B
- 理由：`AGENTS.md` 明确提示 force push、rebase、amend、squash 已发布提交可能导致 Lovable 侧历史丢失
- 影响范围：所有涉及 Git 历史修改的操作
- 是否已被推翻：否
