# 深度对话：非农注水节点证据审计与方法论升级

> 对2025-01至2026-06共18份报告进行方法论2重评后，从“寻找注水”转向“证据分层、节点验证、推理熔断”。

## 主要问题
1. 政府就业、行业集中、修订和政治背景直接相加，产生不可复现的注水量。
2. 红旗数量被写成注水概率，但没有校准。
3. 总新增接近零时CR3出现超过100%的净分母失真。
4. 从数据异常跨越到蓄意操纵，缺少人为干预证据。

## 跨月事实
- 2025年CES序列相对年度基准存在显著高估。
- 2025年11月受停摆、CPS缺失、低回复率和权重变化影响。
- 2026年2月医疗罢工、3月返岗形成一次性V形。
- 工资、工时、参与率和行业集中只能判断增长质量。
- 现有材料没有提供BLS蓄意操纵的直接证据。

## 推理边界
| 观测 | 最多推出 | 禁止推出 |
|---|---|---|
| 回复率下降 | 不确定性上升 | 必然高估 |
| 连续下修 | 初值方向性偏差 | 人为篡改 |
| 基准下修 | 样本序列高估 | 政治操纵 |
| 政府贡献高 | 财政托底 | 虚假岗位 |
| 行业集中 | 增长狭窄 | 岗位不存在 |
| 工资工时弱 | 就业质量降温 | headline造假 |
| 参与率下降 | 分母收缩风险 | 失业率被篡改 |
| 罢工返岗 | 一次性恢复 | 注水 |
| 政治敏感期 | 动机先验 | 已发生干预 |

## 已采纳
数据节点、统计机制、结构贡献、劳动质量、事件配对、三轨判断、证据等级和熔断。

## 待证
H01初值乐观偏差；H02出生—死亡高估；H03季调异常；H04非回复方向偏差；H05政府叙事托底；H06行业集中持续化；H07修订方向集群；H08外部干预。

H01-H07最多证明系统性高估或结构偏差；只有H08取得文件、指令、证词、审计或可复现人工修改，才可能通向蓄意操纵。

## 决定
- 冻结主动注水概率和“真实就业=初值-注水量”。
- 政治背景只作先验倾斜。
- CR3失效时改用正负贡献和扩散指数。
- 可疑节点必须同时设计确认、证伪和到期关闭。
- 方法论目标改为：明确每项证据最多允许推导到哪里。
