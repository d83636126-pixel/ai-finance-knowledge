## 📚 核心概念索引

### 7项红旗检查

> 见 [[01_方法论/方法论_非农注水识别手册]] 第二节

| 红旗 | 检查内容 | 关键阈值 |
|------|---------|---------|

| 红旗2 | 私营 vs 整体差距 | > 30k 触发 |
| 红旗3 | 时薪同比方向 | 放缓触发 |
| 红旗4 | 周工时变化 | < 34.3h 触发 |
| 红旗5 | 行业CR3 | > 85% 触发 |
| 红旗6 | 前值上修 | > 10% 触发 |
| 红旗7 | 后续下修 | > 15% 确认 |

> 见 [[03_分析案例/5月非农注水与多方共识_2026-07-04]] 第九节

| 方 | 核心动机 |

| 沃什 | 鹰派改革合法性 |
| 华尔街 | 短期套利窗口 |

### 政治压力等级

> 见 [[01_方法论/方法论_非农注水识别手册]] 第三节

| 等级 | 时期 | 预期注水 |
|------|------|---------|
| 🚨 极高 | 大选前2月 / 新主席上任 | +50-80k |
| ❌ 高 | 中期选举前 | +30-50k |
| ⚠️ 中 | FOMC分裂期 | +20-30k |
| ✅ 低 | 常规时期 | < 10k |

### 注水四层次

> 见 [[03_分析案例/5月非农注水与多方共识_2026-07-04]] 第七节

| 层次 | 注水量 | 来源 |

| L1 统计方法 | +2~3万 | 出生-死亡模型 |
| L2 季节性 | +3~4万 | 餐饮酒店旺季 |
| L3 政策驱动 | +5万 | 政府+关税+移民 |
| L4 政治叙事 | +1~2万 | 权力交接期 |

## 🌟 核心贡献

### 1. 建立了完整的"非农注水识别"方法论

- 7项红旗检查（可量化）
- 政治背景评估（可预测）
- 注水量估算模型（可计算）
- 投资决策框架（可操作）

### 2. 验证了5月数据的"注水+合谋"双重特征

- 政府就业占比30% → 🚨
- 私营vs整体差52k → 🚨
- CR3 93% → 🚨
- 后续下修-25% → 🚨

### 3. 揭示了"沃什效应"

- 沃什上任前后3个月（2026.3-5）数据明显注水
- 5月达到注水极值
- 真实就业可能仅+3~7万 vs 公布+17.2万

### 4. 提供了未来半年的数据博弈预测

- 10月注水峰值+80k
- 选举前一周是关键套利窗口
- 11月数据回归，12月基准大下修

### 5. 构建了可复用的知识管理体系

- README索引 + 双向链接
- 每月跟踪模板
- 速查卡片
- 场景化导航
