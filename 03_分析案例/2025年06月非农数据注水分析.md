# 2025年6月非农数据注水分析

> 报告期：2025年6月 | 发布日期：2025年7月3日 | 数据来源：BLS CES

## 一、数据快照

| 指标 | 实际值（初值） | 前值 | 评级 |
|------|--------------|------|------|
| 新增非农就业 | **+14.7万** | +14.4万（5月） | 反弹叙事 |
| 失业率 | **4.1%** | 4.2% | 意外下降 |
| 私营部门就业 | **+7.4万** | +13.7万 | **远低于整体** |
| 政府部门就业 | **+7.3万** | +0.7万 | **州教育驱动** |
| 平均时薪（年率） | **+3.7%** | +3.9% | **放缓** |
| 平均每周工时 | **34.2小时** | 34.3小时 | 下降 |
| 劳动参与率 | **62.3%** | 62.4% | 下降 |
| 沮丧工人 | **+25.6万** | — | 大幅上升 |

**即时判断**：headline反弹至+14.7万，但政府占新增 **49.7%** 是致命红旗。

## 二、7项红旗检查

| 红旗 | 计算 | 评级 |
|------|------|------|
| ① 政府占比 | 7.3÷14.7 = **49.7%** | 🚨 |
| ② 私营vs整体差 | **7.3万** | 🚨 |
| ③ 时薪同比 | 3.9%→3.7% 放缓 | ❌ |
| ④ 周工时 | 34.3→34.2h 下降 | ❌ |
| ⑤ CR3 | 政府7.3+医疗3.9+社会1.9 = **89%** | 🚨 |
| ⑥ 前值下修 | 3-4月已大幅下修 | ❌ |
| ⑦ 后续下修 | 8月 **+1.4万**；最终 **-1.3万** | 🚨🚨 |

**合计：6/7 → 严重注水（2025年最严重月份之一）**

## 三、行业拆解

| 行业 | 新增(千) | 备注 |
|------|---------|------|
| **州政府** | **+47** | 教育 **+40**（暑期招聘） |
| **地方政府教育** | **+23** | 季节性 |
| 医疗保健 | +39 | 低于5月 |
| 社会援助 | +19 | — |
| 联邦政府 | **-7** | 累计 **-69** |
| 休闲酒店 | +20 | 低于5月 |
| 制造业/零售/批发 | 变化不大 | 广泛停滞 |

## 四、宏观交叉验证

| 指标 | 信号 | 一致？ |
|------|------|--------|
| 失业率下降 | 4.2%→4.1% | ✅ |
| 参与率 | 62.4%→62.3% | ❌ |
| 就业人口比 | 59.7% 持平 | ❌ |
| 沮丧工人 | +25.6万 | ❌ |
| 边缘劳动力 | +23.4万 | ❌ |
| 时薪/工时 | 双弱 | ❌ |

**6项中仅1项一致 → 注水概率极高**

## 五、注水量估算

| 来源 | 估算(k) |
|------|--------|
| 政府季节性（教育） | **+50~60** |
| 统计方法 | +20~30 |
| 叙事注水（5月疲软后需反弹） | +20~30 |
| **合计注水** | **+90~120** |
| **真实就业** | **+30~50**（最终：**-13**） |

## 六、政治背景

- 5月疲软后市场需要「复苏」叙事
- 6月为暑期教育招聘旺季
- 压力等级：🚨 极高

## 七、综合研判

| 项目 | 判断 |
|------|------|
| 注水概率 | **>80%** |
| 真实就业 | **负/零增长**（最终-1.3万） |
| 修正路径 | +14.7万 → +1.4万（8月） → **-1.3万**（基准后） |
| 操作含义 | 不应被初值反弹误导；T+30大幅下修是确定性事件 |
