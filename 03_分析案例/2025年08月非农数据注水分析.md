# 2025年8月非农数据注水分析

> 报告期：2025年8月 | 发布日期：2025年9月5日 | 数据来源：BLS CES

## 一、数据快照

| 指标 | 实际值（初值） | 前值（修正后） | 评级 |
|------|--------------|--------------|------|
| 新增非农就业 | **+2.2万** | +7.9万（7月） | **极弱** |
| 失业率 | **4.3%** | 4.2% | 2021年10月以来高位 |
| 私营部门就业 | **+3.8万** | +7.7万 | — |
| 政府部门就业 | **-1.6万** | +0.2万 | 联邦 **-1.5万** |
| 平均时薪（年率） | **+3.7%** | +3.9% | 放缓 |
| 平均每周工时 | **34.2小时** | 34.2小时 | 持平 |
| 劳动参与率 | **62.3%** | 62.2% | 同比-0.4ppt |
| 长期失业 | **190万** | — | 同比+38.5万 |

**即时判断**：就业引擎熄火；同日预告9月9日基准修订。

## 二、7项红旗检查

| 红旗 | 计算 | 评级 |
|------|------|------|
| ① 政府占比 | 政府 **-1.6万** | ✅ |
| ② 私营vs整体差 | **-1.6万** | ✅ |
| ③ 时薪同比 | 3.9%→3.7% 放缓 | ❌ |
| ④ 周工时 | 34.2h 持平 | ⚠️ |
| ⑤ CR3 | 医疗3.1+社会1.6 = 约 **85%** | ❌ |
| ⑥ 前值下修 | 6月 **-1.3万**（转负） | 🚨 |
| ⑦ 后续下修 | 最终 **-2.6万** | 🚨 |

**合计：4/7 → 数据可信度高于上半年，但经济本身极弱**

## 四、重大背景：基准修订预告

| 事件 | 说明 |
|------|------|
| 9月9日基准初值 | 截至2025年3月12个月非农下修 **-91.1万**（BLS史上最大） |
| 6月最终值 | 三修后 **-1.3万**（首次转负） |
| 8月自身修订 | 初值+2.2万 → 最终 **-2.6万** |

## 六、政治背景

- 9月17日FOMC将首次降息（后确认-25bp）
- 就业放缓为降息提供叙事支撑
- 压力等级：⚠️ 中（数据已无法「做大」）

## 七、综合研判

| 项目 | 判断 |
|------|------|
| 注水概率 | **60~70%**（初值仍略偏高） |
| 真实就业 | **负/零** |
| 关键信号 | 上半年「强劲」叙事在此月彻底崩塌 |
| 投资含义 | 弱数据+降息预期 → 债市牛陡 |
