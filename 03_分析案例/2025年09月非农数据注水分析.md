# 2025年9月非农数据注水分析

> 报告期：2025年9月 | 发布日期：2025年11月20日（延迟6周+） | 数据来源：BLS CES

## 一、数据快照

| 指标 | 实际值 | 前值（修正后） | 评级 |
|------|--------|--------------|------|
| 新增非农就业 | **+11.9万** | -0.4万（8月） | 表面反弹 |
| 失业率 | **4.4%** | 4.3% | **2025年新高** |
| 私营部门就业 | **+9.7万** | +1.8万 | — |
| 政府部门就业 | **+2.2万** | -2.2万 | 联邦 **-0.3万** |
| 平均时薪（年率） | **+3.8%** | +3.7% | 持平 |
| 平均每周工时 | **34.2小时** | 34.2小时 | 持平 |
| 劳动参与率 | **62.4%** | 62.3% | 同比持平 |
| 就业人口比 | **59.7%** | 59.6% | 同比-0.4ppt |

**即时判断**：延迟发布+收集率异常（80.2%），数据质量打折。

## 二、7项红旗检查

| 红旗 | 计算 | 评级 |
|------|------|------|
| ① 政府占比 | 2.2÷11.9 = **18.5%** | ⚠️ |
| ② 私营vs整体差 | **2.2万** | ⚠️ |
| ③ 时薪同比 | 3.7%→3.8% | ⚠️ |
| ④ 周工时 | 34.2h 持平 | ⚠️ |
| ⑤ CR3 | 医疗4.3+餐饮3.7+社会1.4 = **79%** | ⚠️ |
| ⑥ 前值下修 | 7-8月合计 **-3.3万** | ⚠️ |
| ⑦ 后续下修 | 最终 **+10.8万**（-1.1万） | ⚠️ |

**合计：3~4/7 → 中等水分，但数据可靠性存疑**

## 四、特殊因素：政府停摆

| 因素 | 影响 |
|------|------|
| 发布延迟 | 正常周五发布 → 延迟6周至11月20日 |
| 收集率 | 机构调查80.2%，高于通常水平 |
| 10月数据 | **未发布**（与11月合并发布） |
| 可信度 | 停摆前后统计口径变化，交叉验证困难 |

## 五、注水量估算

| 来源 | 估算(k) |
|------|--------|
| 季节性 | +30~40（餐饮+37） |
| 统计方法 | +10~20 |
| 停摆噪音 | 不可量化 |
| **合计注水** | **+40~60** |
| **真实就业** | **+60~80**（修订后约+108） |

## 六、政治背景

- 9月17日FOMC **首次降息-25bp**（4.00-4.25%）
- 理由：「通胀进展+劳动力市场放缓」
- 压力等级：⚠️ 中

## 七、综合研判

| 项目 | 判断 |
|------|------|
| 注水概率 | **45~55%** |
| 真实就业 | 弱中偏强（餐饮季节性） |
| 关键信号 | 失业率4.4%创年内新高，降息逻辑强化 |
| 数据警示 | **停摆月数据不应与常规月直接比较** |
