# 2025年11月非农数据注水分析

> 报告期：2025年11月 | 发布日期：2025年12月16日 | 数据来源：BLS CES

## 一、数据快照

| 指标 | 实际值（初值） | 1月修订 | 评级 |
|------|--------------|---------|------|
| 新增非农就业 | **+6.4万** | **+5.6万** | 弱 |
| 失业率 | **4.6%** | — | 维持高位 |
| 私营部门就业 | **+6.9万** | — | — |
| 政府部门就业 | **-0.5万** | — | 联邦 **-0.6万** |
| 平均时薪（年率） | **+3.5%** | — | **明显放缓** |
| 平均每周工时 | **34.3小时** | 34.2小时 | 微升 |
| 劳动参与率 | **62.5%** | — | — |
| 经济原因兼职 | **+90.9万**（vs 9月） | — | 恶化 |

**即时判断**：停摆后首份「正常」非农，但仍受低回复率（64.0%）影响。

## 二、7项红旗检查

| 红旗 | 计算 | 评级 |
|------|------|------|
| ① 政府占比 | -0.5÷6.4 = 负值 | ✅ |
| ② 私营vs整体差 | **-0.5万** | ✅ |
| ③ 时薪同比 | **+3.5%** 全年低点附近 | ❌ |
| ④ 周工时 | 34.2→34.3h | ⚠️ |
| ⑤ CR3 | 医疗6.4+建筑2.8+社会1.8 = **约85%** | ❌ |
| ⑥ 前值下修 | 8-9月合计 **-3.3万** | ⚠️ |
| ⑦ 后续下修 | -0.8万（64→56） | ✅ 小幅 |

**合计：3/7 → 中等水分，经济本身偏弱**

## 三、行业拆解

| 行业 | 新增(千) | 备注 |
|------|---------|------|
| 医疗保健 | +46 | 门诊+24，医院+11 |
| 建筑业 | **+28** | 非住宅专业贸易+19 |
| 社会援助 | +18 | — |
| 交通仓储 | **-18** | 快递-18，2月以来累计-78 |
| 休闲酒店 | **-12** | 逆转 |
| 联邦政府 | **-6** | 10月已-162，累计 **-271** |
| 临时帮助 | -5.0 | 持续负增长 |

## 四、停摆残留影响

| 因素 | 说明 |
|------|------|
| 家庭调查 | 10月缺失，11月用2个月合成权重 |
| 回复率 | 64.0%，低于正常 |
| 统计显著性 | 失业率变动需0.26ppt（通常0.21ppt） |
| 10月政府 | -162k为买断计划+停摆叠加 |

## 五、注水量估算

| 来源 | 估算(k) |
|------|--------|
| 统计方法 | +10~20 |
| 医疗+建筑集中 | +10 |
| 停摆噪音 | 不可量化 |
| **合计注水** | **+20~30** |
| **真实就业** | **+30~40**（修订后约+56） |

## 六、政治背景

- 11月无FOMC
- CPI 11月 **2.7%**，核心 **2.6%**（2021年3月以来最低）
- 压力等级：⚠️ 中

## 七、综合研判

| 项目 | 判断 |
|------|------|
| 注水概率 | **40~50%** |
| 真实就业 | 弱 |
| 关键信号 | 时薪同比降至3.5%；交通仓储持续失血 |
| 投资含义 | 弱就业+低通胀 → 降息周期延续 |
