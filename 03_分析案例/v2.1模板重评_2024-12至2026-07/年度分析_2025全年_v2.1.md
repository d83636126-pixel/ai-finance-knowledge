# 年度分析：2025年非农（模板 v2.1）

> 方法论：[[01_方法论/月度跟踪_非农分析模板_v2.1]]  
> 覆盖：2025-01 → 2025-12（12 个月）  
> 整理日期：2026年7月15日

## 一、年度核心结论

2025年主结论不是「多数月份被主动注水」，而是：**初值在就业转弱阶段系统性高估；年度基准修订确认全年接近停滞；下半年叠加停摆制度冲击与增长狭窄化。现有材料没有蓄意操纵的具体证据。**

## 二、总量账本

| 指标 | 数值 |
|------|------|
| 初值合计 | **+1208k** |
| 最新修订合计 | **+181k** |
| 高估缺口（初值−最新） | **+1027k** |
| 月均最新 | **+15k** |

## 三、分季摘要

| 季度 | 判断 |
|------|------|
| 2025Q1 | 初值偏强，基准后接近停滞 |
| 2025Q2 | 进一步降温；6月政府托底特征突出 |
| 2025Q3 | 停滞显性化；8月拐点月 |
| 2025Q4 | 停摆扭曲10月；11–12月弱企稳 |

## 四、双列模型年度口径

| 列 | 年度解读 |
|----|----------|
| 主动注水 | **不支撑「全年系统性蓄意操纵」叙事**；单月政府占比/叙事压力仅作 prior tilt |
| 系统偏差 | **主因**：Early Estimate Bias、SAF/出生死亡、回复率、停摆缺口、人口控制机械位移 |
| 熔断提示 | 若注水点估计与基准修订方向连续相反，冻结精确注水量，只保留结构异常清单（本批次已遵守） |

## 五、C轨压力指数年度回顾

用途：提高质量警惕、缩短持仓 horizon，**不是有罪认定**。  
2025：上半年修订压力高、下半年制度冲击高。  
2026H1：事件驱动波动（罢工）+ 5月集中度压力。

## 六、投资决策年框（序数警示）

| 维度 | 结论 |
|------|------|
| 就业趋势 | 停滞/极慢扩张 |
| 交易栏 | 禁止把年度「注水%」写入仓位公式；绑定 CPI、利率路径与下一次基准修订 |
| 失效条件 | 样本外信息比≤0 连续两季 → 框架降级为阅读清单 |
