## 一、一句话

**数字多半还在，指控链条变了。**  
旧版习惯用「注水概率 + 红旗合计」暗示数据不可信甚至接近操纵；v2.1 把同一批异常拆进 **系统偏差 / 结构狭窄 / 制度冲击**，并在 B 轨写死：**无蓄意操纵具体证据 → 不得升格**。

## 三、框架层：改了什么 / 没改什么

### 改了什么（高影响）

| 维度 | 旧版 | v2.1 重评 |
|------|------|-----------|
| 红旗 | 常加总 n/7 → 直接出「可信/注水」 | **只做初筛**，禁止异质加总综合判定 |
| 注水% | 常被读成可交易概率 | 标明**修辞装置**；交易改**序数警示 + 失效条件** |
| 双列 | 仅早期少数月明确主动/系统 | **全样本双列 + 去重核对** |
| 操纵 | 高% / 高红旗易滑向「有人动手」 | **B轨证据门槛**；无证据不升格 |
| 修订 | 时有、口径不一 | 每月强制 **初值→最新** + H01/H07 台账 |
| 新模块 | 少 | H04/H10 回复率与缺口、H09 人口控制、A轨六机制子表、反方论证 |

### 没改什么（连续点）

1. **停摆月（2025-10）**：剥离政府制度冲击后再谈私营趋势——前后一致。  
2. **真弱月（2026-02）**：低注水 / 真疲软——同向。  
3. **结构事实**：政府占比异常、CR3 集中、参与率假改善等现象多数被保留，变的是**归因轨道**。

### 仍待双方补齐

CES 回复率时序、JOLTS/初请/ISM 全样本、T+0 行情回填；部分 Street 共识仍为近似值。

## 五、冲突最大的 4 个月（优先看）

### 1) 2025-05：旧「70~80%注水」 vs 新「无操纵证据 + 终值+13k」
- **保留**：集中度/质量差。  
- **删除**：把高红旗直接换成操纵指控。  
- **新读法**：初值叙事失效，以修订后停滞为准。

### 2) 2025-06：旧「>80% / 政府托底最严重」 vs 新「结构问题 + 终值-20k」
- **保留**：政府占比异常这一事实。  
- **改写**：从「主动注水杰作」→「托底结构 + 系统修订后转负」。

### 3) 2026-05：旧「>70% 注水爆款」 vs 新「狭窄增长 + 下修至+129k」
- **保留**：CR3/行业集中观察。  
- **改写**：爆款指控降级为结构异常清单与 C 轨压力。

### 4) 2025-03：旧「爆款月 55~65%」 vs 新「修订后+67k、主动列大幅下降」
- 典型「标题迷信」被事后修订证伪；新稿用修订锚定，不再维持爆款注水主线。

## 七、读写仲裁规则（给后续使用）

| 场景 | 优先引用 |
|------|----------|
| 要填空模板、台账、H0X、季度/年度 | **v2.1 重评** |
| 要一句话回答「有没有操纵证据」 | **方法论2全量重评** + v2.1 B轨 |
| 要当时措辞强度 / 行业细表语感 | 旧版单月（**史料**，勿当仓位概率） |
| 旧高% 与 新「无操纵证据」冲突 | **以 v2.1 / 方法论2 为准** |

**禁止**：把旧版注水% 与新版修辞% 做简单相减当「进步幅度」——尤其在 thin print 月（如 2025-08），小分母会把修辞% 拉爆。
