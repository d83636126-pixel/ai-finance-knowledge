# 2026年4月非农数据注水分析

> 报告期：2026年4月 | 发布日期：2026年5月8日 | 数据来源：BLS CES

## 一、数据快照

| 指标 | 实际值（初值） | 修订后 | 评级 |
|------|--------------|--------|------|
| 新增非农就业 | **+11.5万** | **+14.8万**（后续修订） | 超预期（预期+6.2万） |
| 失业率 | **4.3%** | 4.3% | 持平 |
| 私营部门就业 | **+12.3万** | **+14.8万** | — |
| 政府部门就业 | **-0.8万** | **-0.8万** | 联邦 **-0.9万** |
| 平均时薪（年率） | **+3.6%** | — | **放缓** |
| 平均每周工时 | **34.3小时** | — | 微升 |
| 劳动参与率 | **61.8%** | — | ↓0.1ppt |
| 经济原因兼职 | **+44.5万** | — | 恶化 |

**即时判断**：温和正增长，但参与率持续下行+通胀升温，「滞胀」信号浮现。

## 二、7项红旗检查

| 红旗 | 计算 | 评级 |
|------|------|------|
| ① 政府占比 | -0.8÷11.5 = 负值 | ✅ |
| ② 私营vs整体差 | **-0.8万** | ✅ |
| ③ 时薪同比 | 3.5%→3.6% 仍偏低 | ⚠️ |
| ④ 周工时 | 34.2→34.3h | ✅ |
| ⑤ CR3 | 医疗3.7+交通3.0+零售2.2 = **7.9万/11.5万 = 69%** | ⚠️ |
| ⑥ 前值下修 | 2月 → **-15.6万** | ❌ |
| ⑦ 后续修订 | 待5-6月报告验证 | 待验证 |

**合计：3/7 → 中等水分**

## 三、行业拆解

| 行业 | 新增(千) | 备注 |
|------|---------|------|
| 医疗保健 | +37 | 护理+15，家庭健康+11 |
| 交通仓储 | **+30** | 快递+38（反弹） |
| 零售贸易 | **+22** | 仓储会员店+18 |
| 社会援助 | +17 | 个人/家庭服务+24 |
| 信息业 | **-13** | 电信/影视持续萎缩 |
| 联邦政府 | **-9** | 部分停摆仍计就业 |
| 建筑业/制造业 | 变化不大 | 高利率压制 |
| 休闲酒店 | +14 | — |

## 四、宏观环境（本月特殊）

| 指标 | 数值 | 信号 |
|------|------|------|
| CPI年率 | **3.8%** | 能源冲击 |
| 核心CPI | 2.8% | 温和 |
| PPI同比 | **6.0%** | 2022年12月以来最高 |
| 密歇根信心 | **44.8** | 历史最低 |
| 1年通胀预期 | 4.8% | 极高 |

> 就业温和 + 通胀飙升 + 信心崩盘 = **滞胀交易**窗口。

## 六、政治背景

- 4月无FOMC；利率维持3.50-3.75%
- 政府部分停摆期间联邦雇员仍计为就业
- 沃什鹰派立场 vs 疲软参与率
- 压力等级：❌ 高

## 七、综合研判

| 项目 | 判断 |
|------|------|
| 注水概率 | **45~55%** |
| 真实就业 | 中等偏弱 |
| 3个月均值 | **约+4.8万/月**（1-4月修订后） |
| 关键信号 | 参与率连降4个月；通胀与就业脱钩 |
| 投资含义 | 黄金+长债受益；勿因+11.5万忽视滞胀 |
