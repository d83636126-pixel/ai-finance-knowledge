# BLS 非农官方数据抓取工具

> 工具定位：这是一个独立数据抓取工具，可被多个项目复用；financial-alert-system 只是当前接入方之一。

## 一句话说明

这是一个独立的官方非农就业数据抓取工具。主用方式是抓取 BLS Employment Situation 新闻稿全文；辅助方式是通过 BLS Public Data API 抓取单条时间序列。它可以被金融图谱、宏观研究、数据复核、非农分析模板等多个场景复用。

注意：非农就业数据的官方发布机构是 **BLS（U.S. Bureau of Labor Statistics，美国劳工统计局）**，不是美联储。美联储会参考、讨论或回应这些数据，但官方数据源是 BLS。

## 新闻稿全文抓取（当前主用）

如果目标是抓取 BLS 新闻稿页面里的全部发布内容，例如：

```text
https://www.bls.gov/news.release/archives/empsit_03072025.htm
```

应使用新闻稿全文抓取器，而不是单条时间序列 API。现在主用方式是只输入数据年月，让工具自己去 BLS 官方归档中查找网页：

```powershell
cd F:\financial-alert-system
node .\scripts\fetch_bls_empsit_release.js --year 2025 --month 2
```


多个月份也可以一次抓取：

```powershell
node .\scripts\fetch_bls_empsit_release.js --year 2025 --months 2,3,4,5,6
```

这会分别输出每个月对应的官方新闻稿 JSON 和 Markdown。
月份范围也支持：

```powershell
node .\scripts\fetch_bls_empsit_release.js --year 2025 --months 2-6
node .\scripts\fetch_bls_empsit_release.js --year 2025 --months Feb-Jun
```
工具会自动验证页面标题，例如：

```text
THE EMPLOYMENT SITUATION -- FEBRUARY 2025
```

也可以手动提供 URL：

```powershell
node .\scripts\fetch_bls_empsit_release.js --url https://www.bls.gov/news.release/archives/empsit_03072025.htm
```

默认输出到 Obsidian：

```text
F:\AI 金融知识点\02_数据\数据工具\BLS非农数据\news_releases\empsit_03072025.json
F:\AI 金融知识点\02_数据\数据工具\BLS非农数据\news_releases\empsit_03072025.md
```

这类输出包含：

- 新闻稿正文
- embargo / USDL 编号 / 发布日期
- Household Survey Data
- Establishment Survey Data
- Summary table A / Summary table B
- Table A-1 到 A-16
- Table B-1 到 B-9
- 脚注和说明
- `full_text` 全文块

已验证示例：

```text
THE EMPLOYMENT SITUATION -- FEBRUARY 2025
USDL-25-0296
release date: 2025-03-07
检测到表格块：27
```

如果 BLS 拒绝自动访问，可在浏览器打开页面并另存为 HTML，然后执行：

```powershell
node .\scripts\fetch_bls_empsit_release.js --input-html empsit_03072025.htm
```


## 中文翻译功能

建议保留官方英文原文，再额外生成中文翻译版：

```powershell
$env:OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
node .\scripts\fetch_bls_empsit_release.js --year 2025 --month 2 --translate zh
```

输出示例：

```text
empsit_03072025.json
empsit_03072025.md
empsit_03072025_中文翻译.md
```

也可以只翻译已经抓好的 JSON：

```powershell
$env:OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
node .\scripts\translate_bls_release.js --input "F:\AI 金融知识点\02_数据\数据工具\BLS非农数据\news_releases\empsit_03072025.json" --lang zh
```

注意：

- `OPENAI_API_KEY` 只在翻译时需要。
- 英文原文和 JSON 是证据源，中文翻译只是阅读层。
- 翻译文件不会覆盖英文原文。
使用 Claude Code 中转或其它 OpenAI-compatible 中转：

```powershell
$env:CLAUDE_CODE_API_KEY="YOUR_RELAY_KEY"
$env:CLAUDE_CODE_BASE_URL="https://your-relay.example.com"
$env:CLAUDE_CODE_MODEL="claude-3-5-sonnet-latest"
node .\scripts\fetch_bls_empsit_release.js --year 2025 --month 2 --translate zh --provider claude-code
```

中转接口应兼容 OpenAI chat completions。如果 base URL 是：

```text
https://your-relay.example.com
```

工具会调用：

```text

```
## 时间序列 API 抓取（辅助）

`fetch_bls_nonfarm.js` 只抓取单条 BLS 时间序列，例如 `CES0000000001`。它适合做结构化指标跟踪，但不是完整新闻稿。
## 默认抓取的数据

默认 BLS 序列：

```text
CES0000000001
```

含义：

- 指标：Total nonfarm payroll employment
- 来源：BLS Current Employment Statistics (CES)
- 频率：月度
- 单位：千人
- 口径：季调

## 基础用法

抓取默认近几年数据，并默认保存到 Obsidian：

```powershell
node .\scripts\fetch_bls_nonfarm.js
```

抓取指定年份：

```powershell
node .\scripts\fetch_bls_nonfarm.js --start-year 2026 --end-year 2026
```

抓取指定年份区间：

```powershell
node .\scripts\fetch_bls_nonfarm.js --start-year 2020 --end-year 2026
```

如需临时改存其它位置，可如需临时改存其它位置，可指定输出文件：

```powershell
node .\scripts\fetch_bls_nonfarm.js --start-year 2026 --end-year 2026 --out bls_nonfarm_2026.json
```

## 月份参数

抓取后只保留 1 月：

```powershell
node .\scripts\fetch_bls_nonfarm.js --start-year 2026 --end-year 2026 --month 1
```

抓取后只保留 1 月和 2 月：

```powershell
node .\scripts\fetch_bls_nonfarm.js --start-year 2026 --end-year 2026 --months 1,2
```

支持的月份写法：

```text
1
01
M01
Jan
January
```

多个月份也支持英文：

```powershell
node .\scripts\fetch_bls_nonfarm.js --start-year 2026 --end-year 2026 --months Jan,Feb,Dec
```

实现逻辑：先按年份范围完整抓取 BLS 数据，再在本地按月份筛选。这样 `month_over_month_change` 仍然基于真实相邻月份计算，而不是基于筛选后的列表重新计算。

## API Key

小规模请求可以不填 BLS key。

如果有 BLS registration key，可以用参数传入：

```powershell
node .\scripts\fetch_bls_nonfarm.js --api-key YOUR_BLS_KEY
```

也可以用环境变量：

```powershell
$env:BLS_API_KEY="YOUR_BLS_KEY"
node .\scripts\fetch_bls_nonfarm.js
```

脚本也兼容：

```text
BLS_REGISTRATION_KEY
```

## 输出结构

输出 JSON 主要包含：

```text
source       数据源和序列说明
query        查询年份、筛选月份、是否使用 key
latest       筛选后最新一条数据
previous     筛选后上一条数据
observations 所有筛选后的观测值
```

单条观测示例：

```json
{
  "date": "2026-02-01",
  "year": 2026,
  "period": "M02",
  "period_name": "February",
  "value": 158436,
  "latest": false,
  "footnotes": [],
  "month_over_month_change": -156
}
```

解释：

- `value`：非农就业总量，单位是千人。
- `month_over_month_change`：环比变化，单位也是千人。
- `footnotes`：BLS 脚注，例如 `P: preliminary` 表示初值。
- BLS 会修订历史数据，因此重大分析前应重新抓取。
