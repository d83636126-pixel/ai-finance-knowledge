# 非农数据注水识别 · 速查卡片

## 🎯 一句话结论

> 5月非农 +17.2万 的"翻倍超预期" ≈ 真实就业 ~+3~7万 + 注水 ~+10~14万

## 📊 注水四个层次

| 层次 | 注水量 | 来源 |
|------|--------|------|
| L1 统计方法 | +2~3万 | 出生-死亡模型、基准修订 |
| L2 季节性 | +3~4万 | 餐饮酒店旺季 |
| L3 政策驱动 | +5万 | 政府+关税+移民 |
| L4 政治叙事 | +1~2万 | 权力交接期 |
| **合计** | **~+10~14万** | |

## 🔍 5个交叉验证指标

| 指标 | 5月 | 信号 |
|------|-----|------|
| 失业率 | 4.3% 持平 | ✅ |
| 参与率 | 61.8% 持平 | ⚠️ |
| 周工时 | 34.3小时 持平 | ❌ |
| 时薪同比 | +3.4%↓ | ⚠️ |
| 私营 vs 整体差 | 5.2万 | ❌（政府占30%）|

**判断公式**：4个红旗同时出现 → 数据大概率注水

## 🏛️ 多方合谋图谱

| 方 | 收益 |
|----|------|
| 特朗普 | 大选叙事 |
| 沃什 | 鹰派改革窗口 |

## ⚠️ 关键预警信号

- 大选周期 = 数据放大周期
- 2026年7-11月（中期选举前）几乎每月都可能被放大
- 政府就业占比 > 25% 是强红旗
- 后续下修幅度 > 15% 确认注水

## 🛠️ 实战工具

每次非农数据公布后，按以下顺序检查：

1. [ ] 政府就业占比（>25% 红旗）
2. [ ] 私营 vs 整体差距（>3万 红旗）
3. [ ] 时薪同比是否放缓
4. [ ] 周工时是否持平或下降
5. [ ] 行业分布是否过度集中（前三行业 >80% 红旗）
6. [ ] 前2个月是否被大幅上修
7. [ ] 后续报告是否大幅下修（验证用）

**满足4项以上 → 注水概率 >70%**
