## 八、投资决策

### 决策矩阵

> **使用说明**：将双重概率合并后查表 —— `综合注水概率 = max(主动注水, 系统偏差) + 0.3 × min(主动注水, 系统偏差)`。主动注水 > 系统偏差时，信任度折扣更大（🚨优先于⚠️）。
> **v2.1新增前置步骤**：合并前先核对第五节"去重核对"列，确认主动注水三个来源（政府就业、行业集中、叙事注水）之间无重复计算，再代入公式。

| 综合注水概率 | 真实就业 | 操作 |
|------------|---------|------|
| < 30% | 强 | ✅ 顺势做多 |
| < 30% | 弱 | ⚠️ 减仓 |
| 30-60% | 强 | ⚠️ 谨慎做多 |
| 30-60% | 弱 | ❌ 做空 |
| > 60% | 强 | ⚠️ 反向操作 |
| > 60% | 弱 | 🚨 大幅做空 |

| 项目 | 判断 |
|------|------|
| 去重核对已完成 | ☐是 ☐否 |
| 主动注水概率 | ___% |
| 系统偏差概率 | ___% |
| 真实就业方向 | ☐强 ☐弱 ☐持平 |
| 推荐操作 | |
| 仓位调整 | ___% |

> **主动注水** = 意图性操纵（政治叙事注水+异常政府就业）
