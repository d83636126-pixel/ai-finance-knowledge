# 方法论：非农注水识别手册（可复用框架）

> 创建日期：2026年7月4日
> 适用场景：每月第一个周五（BLS非农数据发布日）后使用
> 数据源：美国劳工统计局 BLS · 商务部 · 美联储

---

## 🎯 核心理念

> **单月非农数据是"政治化叙事工具"，3个月均值才是"经济真相"。**
> 投资者的工作不是预测非农数据，而是**识别数据中的水分**。

## 📌 三大原则

1. **数据本身是滞后的** —— 真实情况要 6-8 周后才知道
2. **市场反应是即时的** —— 数据公布当下就要决策
3. **合谋是结构性的** —— 大选/政治周期 = 数据放大周期

---

## 🛠️ 第一步：7项红旗检查（每月必做）

### 红旗1：政府就业占比

```
政府就业 = 整体新增 - 私营部门新增
占比 = 政府就业 / 整体新增 × 100%
```

| 占比 | 信号 |
|------|------|
| < 15% | ✅ 正常 |
| 15-25% | ⚠️ 警惕 |
| 25-35% | ❌ 注水红旗 |
| > 35% | 🚨 严重注水 |

**为什么**：政府就业是财政驱动，不可持续。

### 红旗2：私营 vs 整体差距

```
差距 = 整体新增 - 私营部门新增
```

| 差距(万) | 信号 |
|---------|------|
| < 1 | ✅ 正常 |
| 1-3 | ⚠️ 警惕 |
| 3-5 | ❌ 注水红旗 |
| > 5 | 🚨 严重注水 |

### 红旗3：时薪同比方向

| 趋势 | 信号 |
|------|------|
| 同比加速（环比上升） | ✅ 数据可信 |
| 同比持平 | ⚠️ 中性 |
| **同比放缓（环比下降）** | ❌ 注水红旗 |

**逻辑**：就业强 → 劳动力紧张 → 工资上涨。如果就业强但工资放缓，说明就业质量差。

### 红旗4：周工时变化

| 工时 | 信号 |
|------|------|
| 上升（>34.5小时） | ✅ 真实需求 |
| 持平（34.3-34.5） | ⚠️ 中性 |
| **下降（<34.3小时）** | ❌ 注水红旗 |

**逻辑**：工作量不增加但就业人数增加 → 可能是"分蛋糕"。

### 红旗5：行业集中度 CR3 + HHI

```
CR3 = 前3大行业新增 / 总新增 × 100%
HHI = Σ(各行业占比²)    // Herfindahl-Hirschman Index，衡量集中度
```

| CR3 | 信号 |
|-----|------|
| < 60% | ✅ 全面增长 |
| 60-75% | ⚠️ 警惕 |
| 75-85% | ❌ 注水红旗 |
| > 85% | 🚨 严重集中（季节性/单一因素） |

| HHI | 信号 |
|-----|------|
| < 1,000 | ✅ 分散增长 |
| 1,000-1,500 | ⚠️ 正常 |
| 1,500-2,500 | ❌ 中度集中 |
| > 2,500 | 🚨 高度集中 |

**为什么 HHI 比 CR3 更敏感**：CR3 只看前3家，HHI 考虑所有行业分布。当 HHI > 1,500 时，即便 CR3 看似温和，实际增长结构也很脆弱。

**典型注水行业组合**：
- 餐饮 + 政府 + 教育（季节+财政）
- 医疗 + 零售 + 政府（人口+政治）

### 红旗6：前值上修幅度

```
上修幅度 = (新前值 - 原前值) / 原前值 × 100%
```

| 上修幅度 | 信号 |
|---------|------|
| < 5% | ✅ 正常 |
| 5-10% | ⚠️ 警惕 |
| 10-15% | ❌ 注水红旗 |
| > 15% | 🚨 严重注水 |

### 红旗7：后续报告下修（验证用）

```
下修幅度 = (原始值 - 下修值) / 原始值 × 100%
```

| 下修幅度 | 结论 |
|---------|------|
| < 10% | ✅ 数据基本可信 |
| 10-20% | ⚠️ 存在水分 |
| 20-30% | ❌ 明显注水 |
| > 30% | 🚨 严重造假嫌疑 |

**注意**：这个数据要等下一次报告才能验证（滞后 4-5 周）。

## 👁️ 第1.5步：SAF 季节性调整因子专项检查（进阶）

> **适用场景**：1月、2月等季节性最强的月份，或当 CR3 中零售/休闲等季节敏感行业占比异常时。
> **原理**：BLS 使用 X-13ARIMA-SEATS 做季节性调整。若某个行业的 SAF 偏离历史均值，可能暗示调整参数被人为干预。

### 检查清单

| # | 检查项 | 正常区间 | 异常信号 |
|---|--------|---------|---------|
| 1 | **全非农SAF趋势** | 各月因子在历史1σ内 | SAF 出现非连续跳变 |
| 2 | **零售贸易 SAF** | 1月裁员量符合模型预期 | 假日招聘偏弱→裁员偏少→SA自动偏强（机械性偏差） |
| 3 | **净出生-死亡模型** | 各月在历史正常区间（1月约-80~-120k） | 大幅偏离历史均值 |
| 4 | **模型方法变更** | 无变更 | 过渡期混用新旧参数 |

### 零售贸易的机械性偏差（关键警告）

```
假日招聘弱 → 1月裁员少 → 模型"预期"有大裁员但没发生 → SA后自动偏强
```

这是个**技术上的正确结果**，但容易被误读为经济走强。历史上2017年1月、2003年1月均出现过同类模式。**发现零售偏强时，应先排除此机制再指控注水。**

### 如何获取 SAF 数据

BLS 不逐月公开发布 SAF 具体数值，但可以通过以下方式间接判断：
1. 比较 NSA（未季调）与 SA（季调）数据的月度差异
2. 关注 BLS 技术说明中的模型变更公告
3. 对比出生-死亡模型调整值的历史分布

## 📊 第二步：宏观交叉验证矩阵

每次非农数据公布后，与以下 6 个宏观指标交叉验证：

| 验证指标 | 一致 | 矛盾 | 信号 |
|---------|------|------|------|
| **失业率** | 持平/下降 | 上升 | 一致✅ |
| **劳动参与率** | 持平/上升 | 下降 | 一致✅ |
| **JOLTS 职位空缺** | 持平/上升 | 下降 | 一致✅ |
| **初请失业金人数** | 持平/下降 | 上升 | 一致✅ |
| **挑战者裁员数据** | 持平/下降 | 上升 | 一致✅ |
| **ISM 就业分项** | >50 | <50 | 一致✅ |

**判断规则**：
- 6 个指标中 ≥ 5 个一致 → 数据基本可信
- 6 个指标中 3-4 个一致 → 警惕
- 6 个指标中 ≤ 2 个一致 → 注水概率高

### 政治压力等级表

| 时间 | 事件 | 注水压力 |
|------|------|---------|
| 1月20日 | 特朗普就任 | 🚨 极高 |

## 💼 第四步：行业深度拆解

每月必须拆解的 5 个关键行业：

### 行业 1：休闲酒店（季节性最强）

| 子行业 | 关键观察点 |
|--------|----------|
| 餐饮 | 5-8月旺季，2-3月淡季 |
| 住宿 | 6-9月旺季，1-2月淡季 |
| 娱乐 | 5-9月旺季 |

**注水红线**：单月 > +5万

### 行业 2：政府部门（财政驱动）

| 子行业 | 关键观察点 |
|--------|----------|
| 联邦 | 反映DOGE改革进展 |
| 州/地方 | 反映联邦转移支付 |
| 教育 | 4-8月招聘高峰 |

**注水红线**：单月 > +3万

### 行业 3：医疗（结构性稳定）

| 子行业 | 关键观察点 |
|--------|----------|
| 医院 | 老龄化刚需 |
| 护理 | 长期照护需求 |
| 门诊 | 周期性 |

**正常区间**：+2.5~4万/月

### 行业 4：建筑（政策驱动）

| 子行业 | 关键观察点 |
|--------|----------|
| 住宅 | 利率敏感 |
| 非住宅 | 商业周期 |
| 重基建 | 政府投资 |

**政策信号**：基建法案/芯片法案资金拨付期

### 行业 5：金融（领先指标）

| 子行业 | 关键观察点 |
|--------|----------|
| 商业银行 | 信贷周期 |
| 保险 | 利率敏感 |
| 证券 | 市场活跃度 |

**金融业负增长是经济领先指标**：
- 连续2个月负增长 → 警惕衰退
- 单月 <-1.5万 → 风险信号

## 📈 第五步：注水量估算模型（双列升级版）

> v2.0 改进：将单一"注水量"拆分为 **"主动注水"**（意图性操纵）和 **"系统性乐观偏差"**（方法/模型导致的高估）。
> 两者政策含义不同：前者指控政治干预，后者指向统计体系局限。

### 总体公式

```
真实就业 ≈ 整体新增 - （主动注水 + 系统偏差）

主动注水 = 政治叙事注水 + 异常政府就业
系统偏差 = 季节性偏差 + 统计注水 + 人口调整效应 + 回复率偏差
```

### 细项拆分表

> 以下数值为 **2025年1月示例**，实际每月按当下政治背景和数据特征重新估值。

| 来源 | 主动注水(k) | 系统偏差(k) | 性质判断 |
|------|------------|------------|---------|
| **政府就业偏高** | +15~20 | +10 | 政府占比>20%时部分为财政常态化 |
| **行业集中异常** | +0~10 | +15~25 | CR3>75%或HHI>1,500；零售偏高常为SAF机械性偏差 |
| **前值上修溢出** | — | +10 | 基准修订窗口内常规操作，非刻意 |
| **叙事注水** | +10~15 | +5 | 高压期真实存在，但BLS透明度亦需考量 |
| **人口控制调整** | — | +0~15 | 仅家庭调查，机构调查不受影响；但失业率解读需修正 |
| **回复率不足** | — | +5~10 | CES回复率从80%降至~60%，全月适用，非当月独有 |
| **合计** | **+25~40** | **+45~75** | — |

### 各来源估算方法

**主动注水估算**：
```
政府异常 = (当月政府占比 - 15%) × 总新增
叙事注水 = 政治压力等级系数 × 当月新增 × 0.05~0.10
```

**系统偏差估算**：
```
季节性偏差 = Σ(行业SAF异常偏离)    // 需对照历史SAF均值
统计注水 ≈ (前2个月上修合计 + 当月基准调整) × 0.5
回复率偏差 = 基准修订/12          // 年度基准修订分摊到月
人口调整效应 = 仅用于失业率解读    // 见第X步
```

### 注水占比评级

| 主动注水/整体 | 系统偏差/整体 | 综合评级 |
|-------------|-------------|---------|
| < 10% | < 20% | ✅ 健康 |
| 10-20% | 20-30% | ⚠️ 中等 |
| 20-30% | 30-40% | ❌ 明显 |
| > 30% | > 40% | 🚨 严重 |

### 人口控制调整速查

1月是家庭调查（CPS）引入新人口估计的窗口期，影响**不可忽略**：

```
人口调整 = 新估计人口 - 旧估计人口
典型影响：劳动力 +200万，就业 +200万，失业率 +0.1pp

机构调查（CES）不受影响，但失业率解读需在旧口径基础上进行。
```

### 关键区分

| 现象 | 更像主动注水 | 更像系统偏差 |
|------|------------|------------|
| 政府占比>30% | ✅ | — |
| 零售行业异常偏强 | — | ✅ （SAF机械性解释）|
| 后续大幅下修 | — | ✅ （晚期周期偏误+回复率）|
| 政治高压期且多行业异常 | ✅ | — |
| 前值上修温和 | — | ✅ |
| 同月公布大幅基准下修 | ✅ 可能是掩护 | ✅ 更可能是透明度的体现 | 证据双向，需看同期其他信号 |

## 🎯 第六步：投资决策框架

基于注水识别结果，做出投资决策：

### 决策矩阵

> 双概率合并公式：`综合注水概率 = max(主动, 系统) + 0.3 × min(主动, 系统)`
> 主动注水>系统偏差时，信任度折扣更大。

| 综合注水概率 | 真实就业 | 操作 |
|---------|---------|------|
| < 30% | 强 | ✅ 顺势做多 |
| < 30% | 弱 | ⚠️ 减仓 |
| 30-60% | 强 | ⚠️ 谨慎做多 |
| 30-60% | 弱 | ❌ 做空 |
| > 60% | 强 | ⚠️ 反向操作 |
| > 60% | 弱 | 🚨 大幅做空 |

### 套利窗口期

**"先放大、再修正"的时间差**：

```
T+0：非农数据公布（放大版）
T+1 ~ T+5：市场按放大版交易（套利窗口）
T+30 ~ T+45：基准修订（下修）
```

**操作**：
- T+1 ~ T+3：按放大版逻辑做交易
- T+5 ~ T+10：逐步减仓
- T+30：关注修订幅度
