## 🎯 7项红旗检查速记

> **注水识别的7个核心指标：**

```
红旗1：政府就业占比 > 25%         → 🚨
红旗2：私营vs整体差 > 30k         → 🚨
红旗3：时薪同比放缓              → ❌
红旗4：周工时 < 34.3小时          → ❌
红旗5：行业CR3（前3大）> 85%      → 🚨
红旗6：前值上修 > 10%             → ❌
红旗7：后续下修 > 15%             → 🚨
```

**口诀**：政府30、差30、CR3-85、上修10、下修15

## 🚀 5步情景推演法 v2

```
步骤1：定价基准
  └─ 市场共识 + 机构分歧度 + 关键锚定

步骤2：传导链分析
  ├─ 水平信号（level）：绝对水平
  └─ 变化率信号（rate）：边际变化

步骤3：衰减因子
  ├─ 假期流动性
  ├─ 数据质量疑点
  └─ 政治背景

步骤4：情景概率化
  ├─ 情景A（基线）：概率最大
  ├─ 情景B（鸽派）
  └─ 情景C（鹰派）

步骤5：利益方映射
  ├─ 谁受益？
  ├─ 谁受损？
  └─ 内部目标冲突？
```

**口诀**：定价、传导、衰减、情景、利益
