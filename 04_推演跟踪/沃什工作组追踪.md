## 组 1 · 沟通（P2）

| 字段 | 内容 |
|------|------|
| 工作组名称 | 沟通（Communications） |
| 领队及背景倾向 | **Peter R. Fisher / Arminio Fraga / Mervyn King**（官方）；信号锚 King；倾向：呼应沃什「弱化前瞻指引、声明精简」 |
| 预测结论方向（先验） | **鸽/中性偏改革**：建议进一步缩短声明、降低新闻发布会仪式感、弱化路径依赖式指引 |
| 可证伪命题 | ① 年底报告若**反对**已发生的声明精简/指引弱化 → 证伪「背书型」；② 若高度复述已执行操作 → 支持「先斩后奏」 |

| 政府输出 | 与「改革联储、少说多做」议程对齐度 **高** |
| 里程碑时间戳 | 2026-06：声明缩至~130词、删除「进一步调整」类指引（见 [[02_数据/2026年6月美国经济数据汇总]]） |

| 字段 | 内容 |
|------|------|
| 工作组名称 | 资产负债表（Balance Sheet） |
| 领队及背景倾向 | **Karen Dynan / Raghuram Rajan / Jeremy Stein**（官方）；信号锚 Rajan（棘轮效应 / 充裕准备金审慎） |
| 预测结论方向（先验） | **偏鹰/收缩**：支持继续 QT 或更被动的余额表；强调准备金「够用即可」，警惕财政主导叙事 |
| 可证伪命题 | ① 若报告主张**暂停/逆转 QT 并扩表护盘** → 严重背离沃什就职前后缩表叙事；② 若强化 ON RRP/准备金下限讨论 → 部分验证 |

| 政府输出 | 与「降融资成本」诉求可能冲突 → 对齐度 **中偏低**（政治摩擦源） |

## 组 5 · 通胀框架（P4）

| 字段 | 内容 |
|------|------|
| 工作组名称 | 通胀框架（Inflation Framework） |
| 领队及背景倾向 | **Greg Mankiw / Thomas Sargent / William White**（官方）；信号锚 White；锚点：沃什质疑刚性 2% / AIT |
| 预测结论方向（先验） | **慢变量**：年内不大改 2% 数字，但报告将留下「重估 AIT / 灵活度」开口；**不预期**立刻宣布新目标数字 |
| 可证伪命题 | ① 若 **Q3 前**即公开推动重启正式目标讨论 → 偏离「低频」先验，升权为 P1；② 若年底仅技术性肯定现状 → 先验成立 |
| 市场输出 | 长期盈亏平衡通胀、实际利率锚定 |
| 联储内部输出 | 框架审查政治成本高，倾向拖延至选举窗口后 |

| 里程碑时间戳 | 2026-05：沃什就职/听证质疑 2% 刚性（见五月汇总） |
